# MB - Flutter Questions Repository

This repository contains Flutter code examples for various questions (Q1 to Q8b). Each question is implemented in its own separate Dart file.

## Files

- `q1_basic_widgets.dart` - Basic Widgets
- `q2a_column_row_stack.dart` - Column, Row, Stack
- `q2b_layoutbuilder.dart` - LayoutBuilder Responsive
- `q3a_mediaquery.dart` - MediaQuery Responsive
- `q3b_navigation.dart` - Simple Navigation
- `q4a_named_routes.dart` - Named Routes
- `q4b_stateful.dart` - Stateful + Stateless
- `q5_custom_widgets.dart` - Custom Widgets
- `q6a_theme.dart` - Theme Example
- `q6b_form.dart` - Simple Form
- `q7a_validation.dart` - Form Validation
- `q7b_animation.dart` - Size Animation
- `q8b_fade_slide.dart` - Fade & Slide Animation

## How to Run

To run any of these examples, you can copy the code into a new Flutter project and run it using `flutter run`.

## Note

These files contain Dart code snippets for educational purposes. They may require Flutter SDK to compile and run properly.
